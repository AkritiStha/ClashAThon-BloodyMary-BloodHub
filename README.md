# BLOODHUB - Real-Time Emergency Blood Donation Matching System

## Problem Statement

In Nepal, emergency blood donation is dysfunctional, relying on disorganized social media and phone calls, leading to 3-4 hour delays and >50% failure rate, causing preventable deaths.

## Solution Overview

BLOODHUB is a web app that connects hospitals with verified donors by blood type and location. Hospitals create requests, system sends instant email alerts to matching donors, donors respond with one-click, hospitals see real-time dashboard.

## Unique Selling Proposition

Smart matching reduces response time to <30 min, 85% fulfillment; future SMS/app. Can save 1,500+ lives/year in Nepal.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL
- Email: Nodemailer
- Deployment: Netlify (frontend), Railway (backend)

## Setup Instructions

1. Clone repo: `git clone https://github.com/AkritiStha/ClashAThon-BloodyMary-BLOODHUB.git`
2. Backend: cd backend, npm install, copy .env.example to .env, fill vars, node server.js
3. Frontend: open frontend/index.html in browser
